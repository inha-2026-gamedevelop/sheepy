# 보스 기획 (Azathoth)

> 원본 기획 텍스트를 구조화 정리한 문서. 구현과의 불일치/미확정 항목은 `claude/PLAN.md`의 "스펙 불일치 / 기획 확인" 섹션을 참고.

## 공통 규칙

- **총 체력**: 64,000
- **피격 데미지**
  - 보스(본체) 공격을 맞으면 플레이어 하트 **한 칸** 차감
  - 보스 분신 공격을 맞으면 플레이어 하트 **반 칸** 차감
- **페이즈**: 총 4개. 각 페이즈는 체력이 16,000 닳으면(총 체력 대비 페이즈 하한 도달) 피통을 **동결**시키고 해당 페이즈의 종료 기믹을 진행한다
- **보스전 타이머**: 되감기와 무관하게 계속 흐르는 실시간 타이머. **10분(600초) 초과 시 플레이어 즉사**
- **타임 리와인드**: 플레이어가 되감기를 하면 보스 체력(HP)도 함께 되돌아간다

## 공통 패턴 (페이즈 무관)

### 감정 상태

| 감정 | 효과 |
|---|---|
| 검정(Black) | 모든 공격 반사 |
| 하양(White) | 본체(플레이어 본체만) 공격 반사 |
| 남색(Navy) | 분신 공격 반사 |
| 핑크(Pink) | 낙뢰 낙하 비율 x2 |
| 파랑(Blue) | 낙뢰 낙하 비율 x1/2 + 맵에 플레이어 회복용 하트(한 칸) 픽업 제공 |
| 화남(Angry) | 3페이즈 고정 패턴. 10초에 한 번씩 1초간 플레이어 키 반전(혼란), 혼란 상태 아이콘 표시 |

### 낙뢰

- 4초에 한 번씩 맵 위 랜덤 지점에서 아래로 낙하
- 피격 시 하트 한 칸 차감 + 0.5초간 이동 불가(스턴)

## 1페이즈 (분신 독립 피통 - 보스 본체 피통과 분리)

- 보스 분신 2체 등장, 각 분신은 **자기 독립 피통**(`CloneHealth`)을 가진다 - 보스 본체 총 피통을 깎지 않는다 (감정 반사 규칙만 공유)
- 1페이즈에는 보스 총 체력바 대신 분신 2체 개별 체력바를 표시 (분신 체력바에는 25% 페이즈 구분선 없음)
- 분신은 근거리 공격, 애니메이터로 공격 모션 구현
- 분신 2체를 모두 잡으면 종료 기믹 -> 2페이즈 진입 시 보스 본체 체력바가 **75%부터** 표시 (본체 피통은 처음부터 100% 이지만 1페이즈 몫 25%를 분신이 대신 소화한 셈)

> 구현 노트(2026-07-12): 초기 설계는 "각 분신 8,000 = 1페이즈 16,000이 총 64,000의 첫 25%"였으나, 분신 피통을 본체와 완전히 분리하는 방식으로 변경. 본체 바는 페이즈 2~4(75%~0%)만 담당한다
- **종료 기믹 (2페이즈 진입 전 즉사 기믹)**
  1. 맵 전체를 덮는 레이저를 빨강/파랑/초록 중 랜덤 순서로 3회 예고 발사
     - 안전구역은 플레이어가 슬로우모션(느리게) 상태일 때만 표시됨
     - 안전구역은 레이저가 발사될 때까지만 표시
  2. 예고가 끝나고 5초 후, **같은 순서**로 실전 레이저 재발사
     - 예: 예고가 빨강 -> 파랑 -> 초록 순서였다면 실전도 빨강 -> 파랑 -> 초록 순서
     - 발사 시점에 해당 색 안전구역 밖에 있으면 즉사
  3. 파훼 성공 시 보스 본체가 필드에 등장 (2페이즈 시작)

## 2페이즈 (48,000 ~ 32,000)

- 1페이즈 분신의 근거리 공격 패턴 유지 + 맵 아래에서 솟아오르는 **장풍** 패턴 추가
- 두 패턴 모두 애니메이션 보유
- **종료**: 컷신 등장 후 씬 전환으로 맵 변경 (3페이즈 진입)

## 3페이즈 (32,000 ~ 16,000) - 화남 감정 고정

- 2페이즈 패턴 유지 + 맵을 가로지르는 레이저 추가
- 레이저: 발사 전 1.5초간 빨간색 점멸 경고 -> 1초간 발사
- 시작 방향/도착 지점은 랜덤

## 4페이즈 (16,000 ~ 0)

- 타임 리와인드 시스템 삭제(사용 불가)
- 나머지 세부 패턴은 추후 확정
